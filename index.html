Week 6 Day 1 Assignment | Constructors

Instructions:

Part 1

  1. In script.js, create a constructor function named Television.
  2. Add properties to the constructor for power status (on/off) and current channel.
  3. Implement methods using prototypes for turning the TV on and off, which updates the power status.
  4. Add a method to change the channel, updating the relevant property.
  5. Include a method named getState that console.logs the TV's state in a readable format.

Part 2

  1. Instantiate at least three different Television objects with various states and channels.
  2. Use the getState method for each Television instance to display its state in the console.

Part 3

  1. Test each method to confirm that the Television objects are working as expected.
  2. Verify that the getState method outputs the correct status and channel.

Part 4

  1. Commit and push your index.html and script.js files to your GitHub repository.
  2. Deploy your project using GitHub Pages or a similar platform.
  3. Submit the URL link to your constructors GitHub repository and the live site as your assignment submission.
